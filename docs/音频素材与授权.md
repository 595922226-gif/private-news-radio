# 音频素材与授权

当前版本不下载、不使用任何第三方音乐或音效文件。

`scripts/pipeline/sound_design.py` 在每次运行时通过数学波形生成以下原创合成声音：

- `intro.wav`：7 秒晨间开场提示音
- `outro.wav`：8 秒结尾提示音
- `section.wav`：栏目转场音
- `break.wav`：新闻条目分隔音
- `pause.wav`：静音停顿

这些声音由本项目代码即时生成，不包含采样、歌曲、录音或第三方版权素材。最终由 `ffmpeg` 转码、统一响度并与百炼生成的人声合并。

若提示音生成或混音失败，系统会记录警告并退回纯人声音频，保证新闻节目优先生成。
